# Contributing to Postlane Prompts

Prompt contributions are welcome via pull request.

## Acceptance Criteria

All prompt contributions must meet the following criteria:

- **Completeness**: The prompt must include all necessary context, instructions, and examples to achieve its stated goal
- **Platform compliance**: The prompt must adhere to the target platform's API requirements, rate limits, and content policies
- **Voice**: The prompt must maintain Postlane's tone and style guidelines
- **Test evidence**: The contribution must include evidence that the prompt has been tested and produces the expected output

Full contribution criteria and guidelines will be documented as part of Milestone 6.
