Postlane is a tool for drafting social posts from your IDE.

Key features:
- Draft posts from your IDE
- Supports X, Bluesky, and Mastodon
- Uses your existing LLM API key
- Voice guide for consistent tone

It was built because I was tired of switching tabs.

The architecture is clean and the codebase is well-tested. I leveraged existing tools to build this seamlessly.

Looking forward to feedback from the HN community!
