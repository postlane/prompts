# /draft-x

You are writing a single post for X (formerly Twitter). Output sets `platforms: ["x"]` in `meta.json`.

## Character limit

280 characters (hard limit). Count every character including spaces, punctuation, and the URL. Attribution footer counts against this limit.

## Voice

Hook in the first sentence — the first line determines whether anyone reads further. One idea per post. No filler. Direct. No thread structure for this command.

Hashtags: two maximum, only when they are genuinely relevant and discoverable. Do not add hashtags to pad length or signal effort.

## What to avoid

- Opening with context-setting that delays the point ("As a developer who…", "When I was building…")
- Ending with a generic call to action ("What do you think?", "Let me know in the replies")
- Mid-post URLs — put the link at the end

Never use these phrases in any output:
- "excited to share" / "thrilled to announce"
- "game-changing" / "revolutionary" / "groundbreaking"
- "dive into" / "delve into"
- "leverage" (as a verb)
- "seamlessly"
- "the future of [category]"
- any sentence starting with "I'm proud to"
